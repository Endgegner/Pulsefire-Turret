Adds a new maximum range security building to the game:

### The Pulsefire Turret
 
<b>Properties</b>
 - High precision
 - Full Metal Jacket
 - Slow reload
 - 1x1

<b>Requirements</b>
 - Technology "Pulsefire Turret"
   (dependency: "Multi Analyzer")
 - Metal: 125
 - Steel: 125
 - Components: 8
 - Power consumption: 400 W
